# Compute loadings (aka principal components)

Inner function to compute the intercept and loading functions from the
spline coefficient matrix.

## Usage

``` r
compute_loadings(spline.basis, spline.coefficients)
```

## Arguments

- spline.basis:

  The spline basis matrix.

- spline.coefficients:

  the matrix of spline coefficients.

## Value

The matrix of loadings
